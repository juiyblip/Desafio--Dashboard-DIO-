# Desafio--Dashboard-DIO-
Projeto do Santander Bootcamp 2025-DIO
Este projeto faz parte de um desafio do Santander Bootcamp 2025, promovido pela DIO, com foco na criação de um dashboard de vendas no Excel.

O objetivo principal é transformar dados brutos em informações visuais claras e organizadas, permitindo uma análise eficiente do desempenho de vendas e apoiando a tomada de decisões baseada em dados.
